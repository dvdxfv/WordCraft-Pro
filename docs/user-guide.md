# WordCraft Pro 使用指南

> 面向普通用户，无需任何技术知识。打开网站注册登录后直接使用。

---

## 界面概览

打开 WordCraft Pro 后，界面分为三个区域：

- **左侧**：文件列表，显示当前打开的所有文档
- **中间**：文档预览区，查看文档内容
- **右侧**：功能面板，包含格式设置、质量检查、交叉引用三个模块

顶部工具栏提供所有核心操作按钮。

---

## 第一步：上传文档

点击顶部工具栏的 **"打开文件"** 按钮（文件夹图标），选择要检查的文档。

支持的格式：

| 格式 | 说明 |
|------|------|
| .docx | Word 文档（推荐，功能最完整） |
| .doc | 旧版 Word 文档 |
| .pdf | 仅预览，无法做格式检查 |
| .xlsx / .xls | Excel 表格 |
| .txt / .md | 纯文本 |

可以同时打开多个文档，顶部会出现标签页，点击标签页切换文档。

---

## 顶部工具栏按钮说明

| 按钮 | 作用 |
|------|------|
| 返回首页 | 回到登录/首页 |
| 打开文件 | 上传本地文档 |
| 保存 | 保存当前编辑内容 |
| 格式（标尺图标） | 打开右侧格式面板 |
| 检查（感叹号图标） | 打开右侧质量检查面板；数字角标显示问题数量 |
| 交叉引用（链接图标） | 打开右侧交叉引用面板 |
| 导出 | 将文档导出为 Word 文件下载 |

---

## 格式面板

点击顶部"格式"按钮，右侧打开格式面板。格式面板的作用是：**告诉系统这篇文档应该符合什么排版标准**，之后检查时系统就知道哪里不对。

格式面板有两个子标签：**自定义** 和 **AI 智能**。

---

### 格式面板 — 自定义标签

手动填写排版要求。

**预设模板**

最上方有模板下拉菜单，可以直接加载内置模板：
- **海大毕业论文** — 广东海洋大学本科毕业论文格式
- **尖草坪区公文** — 太原市尖草坪区政府公文格式
- **通用实验报告** — 通用实验报告格式

选择模板后，下方所有字段自动填入对应的格式要求，不需要手动填写。

如果有自己保存过的模板，会出现在"我的模板"分组里。

---

**页面设置**

| 字段 | 说明 |
|------|------|
| 纸张 | A4 / A3 / B5 / Letter |
| 上/下 | 页面上下边距，单位 cm |
| 左/右 | 页面左右边距，单位 cm |

---

**标题样式**

可以分别设置一级到四级标题的字体、字号、编号格式：

| 字段 | 说明 |
|------|------|
| 字体下拉 | 黑体 / 宋体 / 楷体 / 仿宋 / 微软雅黑 / Arial / Times New Roman |
| 字号下拉 | 初号到八号，中文字号名称自动换算为磅值 |
| 编号输入框 | 标题编号格式，如"1"、"1.1"、"1.1.1" |

---

**正文样式**

| 字段 | 说明 |
|------|------|
| 西文字体 | 文中英文/数字的字体，Times New Roman / Arial / Calibri |
| 中文字体 | 文中中文的字体 |
| 字号 | 正文字号 |
| 首行缩进 | 每段开头缩进几个字符，通常填 2 |
| 行距 | 行距模式（倍数 / 固定值）+ 数值 |
| 对齐 | 两端对齐 / 左对齐 / 居中 / 右对齐 |

---

**保存为检查规范**（绿色按钮）

填好所有格式要求后，点击这个按钮保存。保存后，质量检查会自动对照这套规范检查文档，发现字体、字号、行距不符合的地方。

> 如果不需要格式检查，可以不填、不保存，质量检查仍然会检查错别字和标点等其他问题。

---

### 格式面板 — AI 智能标签

让 AI 自动读取排版规则，不需要手动填写。

**上传参考模板**

点击"点击上传参考模板"区域，上传一份已经规定了格式要求的文档（.docx 或 .pdf）。AI 会自动读取其中的格式规范。

**快速选择**

下方有四个快速选择按钮：学术论文 / 政府公文 / 实验报告 / 海大论文。点击后会自动填入对应类型的通用描述，再点"AI 解析排版规则"。

**描述框**

也可以直接用中文写出格式要求，例如：
"A4 纸，上下边距 2.5cm，左右 3cm。一级标题黑体三号居中，正文宋体小四首行缩进 2 字符，行距 1.5 倍。"

**AI 解析排版规则**（橙色按钮）

点击后 AI 开始分析，解析结果会显示在下方，可以手动修改。

**保存为检查规范**（绿色按钮，解析完成后出现）

确认解析结果正确后，点击保存，之后质量检查会自动使用这套规范。

---

## 质量检查面板

点击顶部"检查"按钮，右侧打开质量检查面板。

### 检查统计

面板顶部显示四个数字：
- **共计** — 发现的问题总数
- **错误** — 需要修改的明确错误
- **警告** — 可能有问题，需要人工判断
- **提示** — 建议性改进，不强制

### 筛选器

可以按**严重程度**（错误/警告/提示）或**问题类别**（错别字/数据不一致/逻辑问题/格式问题）过滤显示。

### 问题列表

每条问题显示：问题描述、原文位置。每条问题右侧有三个按钮：

| 按钮 | 作用 |
|------|------|
| 定位 | 在中间预览区跳转到该问题所在位置，方便对照查看 |
| 接受 | 接受系统建议的修改，自动替换文中内容 |
| 拒绝 | 忽略这条问题，从列表中移除 |

### 批量操作

问题出现后，顶部会显示三个批量按钮：
- **✓ 全部接受** — 一键接受所有问题的修改建议
- **✗ 全部拒绝** — 一键忽略所有问题
- **↩ 撤销采纳** — 撤销刚才的批量接受操作

### 运行检查按钮

面板底部有两个按钮：

**规则检查**（蓝色，免费）

本地规则引擎，不消耗 AI 额度，即时完成。检查内容：
- 错别字（常见错别字词库 + 的/地/得用法）
- 数据一致性（同一文档中数字、日期、单位前后是否矛盾）
- 标点规范（中英文混排空格、重复标点、括号是否成对）
- 格式合规（字体、字号、行距是否符合已保存的检查规范）

**🤖 AI 深度**（橙色，消耗 AI 额度）

调用 AI 进行深度分析，在规则检查基础上额外检查：
- 论证逻辑是否流畅
- 因果关系是否成立
- 结论是否有依据
- 语法和表述优化建议

## 交叉引用检查面板

点击顶部"交叉引用"按钮，右侧打开交叉引用面板。

### 什么是交叉引用

文档中常会出现"如图 1-1 所示"、"见表 3"、"参见第二章"这类引用。如果图/表/章节编号改了但引用文字没改，或者引用了一个不存在的编号，就是交叉引用错误。

### 检查统计

面板顶部显示五个数字：
- **目标** — 文中找到的图/表/公式/章节目标总数（被引的对象）
- **引用** — 文中找到的引用点总数（"如图 X"这类文字）
- **有效** — 引用与目标正确匹配的数量
- **悬空** — 引用了一个找不到对应目标的编号（危险）
- **未引** — 某个图/表/章节存在，但全文没有地方引用它

### 三个子标签

| 标签 | 内容 |
|------|------|
| 引用目标 | 列出扫描到的所有目标（图/表/公式/章节），含类型、编号、标签、标题 |
| 匹配结果 | 列出所有引用点及其与目标的匹配状态（有效/悬空） |
| 可采纳引用 | 系统建议可以添加或修正的引用，可逐条确认 |

### 运行交叉引用检查（紫色按钮）

点击后系统扫描整篇文档，自动填充上述三个标签的内容。

---

## 导出 Word

点击顶部"导出"按钮，系统将当前文档按照格式面板中设定的排版规则重新生成 Word 文件（.docx）并下载到本地。

中文字体（宋体、黑体等）和英文字体（Times New Roman 等）均正确设置，在 Word 中打开不会出现乱码或字体丢失。

> PDF 文件导出为 PDF，Word 文件导出为 .docx（系统自动判断）。

## 常见问题

**Q：上传文件后文档内容显示不出来**
A：请确认文件格式是否支持（.docx 推荐）。如果是旧版 .doc 文件，可能因系统环境不同无法直接打开，建议先在 Word 中另存为 .docx 格式再上传。

**Q：格式检查没有结果，只有错别字和标点**
A：格式检查需要先在格式面板填好排版要求，并点击"保存为检查规范"，之后运行规则检查才会对照格式校验。

**Q：AI 功能提示"额度不足"**
A：免费账户每日 AI 调用有上限，次日 0 点自动恢复。如果需要更多额度，可升级付费套餐。

**Q：接受了一条修改后发现改错了，怎么撤销**
A：点击顶部编辑工具栏的"↩"（撤销）按钮，或按 Ctrl+Z，可以逐步撤销操作。

**Q：我想用同一套格式规范检查多篇文档**
A：在格式面板设置好规范后，点击"+ 另存为模板"保存到"我的模板"。下次打开新文档后，从模板下拉菜单选择该模板，再运行检查即可。
