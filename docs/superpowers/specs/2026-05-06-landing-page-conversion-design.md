# Landing Page 转化优化设计文档

**日期**：2026-05-06  
**目标文件**：`web/wordcraft_landing.html`  
**核心目标**：同时提升注册量和急诊包购买量（A+C 方向）

## 背景

## 改动三：Pro 方案按钮

**位置**：`#pricing` section 内 Pro 价格卡片底部按钮。

**原代码**：
```html
<button class="price-cta disabled" disabled>即将开放</button>
```

**改为**：
```html
<button class="price-cta primary" onclick="openWechatModal()">微信咨询订阅</button>
```

复用现有 `openWechatModal()` 函数，弹窗内容无需修改。

---

## 改动四：毕业季限时倒计时徽章

**位置**：Pro 价格卡片内 `.price-dot` 徽章处。

**原**：静态文字"毕业季限时"小红点徽章。

**改为**：显示"毕业季限时 · 剩 X 天"，其中 X 由 JS 计算当前日期到 2026-06-30 的剩余天数，每次页面加载时动态渲染。

**实现**：在页面底部 `<script>` 中新增约 5 行倒计时计算逻辑，将结果写入对应 DOM 元素。

## 验收标准

1. 用户评价区在定价区正上方正确渲染，三张卡片横排，移动端自适应折行
2. FAQ 折叠区在 CTA 区正上方，点击问题可展开/收起答案
3. Pro 卡片按钮点击后正确弹出微信咨询弹窗
4. 毕业季倒计时显示正确天数（以 2026-06-30 为截止日，不足 1 天显示 0）
5. 页面整体视觉风格一致，无样式冲突
