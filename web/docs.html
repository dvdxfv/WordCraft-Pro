使用指南

无需任何技术知识，打开网站注册登录后直接使用。

界面概览

进入主界面后，页面分为三个区域：

左侧 — 文件列表
显示当前打开的所有文档，点击标签页切换
中间 — 预览区
显示文档内容，可进行简单编辑
右侧 — 功能面板
格式设置、质量检查、交叉引用，三个模块切换使用

顶部工具栏提供所有核心操作按钮；底部状态栏显示字数和当前状态。

上传文档

点击顶部工具栏的 打开文件 按钮，选择本地文档上传。可以同时打开多个文档，顶部出现标签页，点击标签页切换。

格式支持情况
.docxWord 文档，推荐使用，功能最完整
.doc旧版 Word 文档，支持大部分场景
.pdf支持预览；格式检查功能不适用于 PDF
.xlsx / .xlsExcel 表格，支持预览
.txt / .md纯文本和 Markdown，支持预览

顶部工具栏按钮

按钮作用
返回首页返回登录页 / 首页
打开文件上传本地文档，快捷键 Ctrl+O
保存保存当前编辑内容，快捷键 Ctrl+S
格式（标尺图标）打开右侧格式面板，用于设置排版规范
检查（感叹号图标）打开右侧质量检查面板；图标上的数字显示当前文档问题数量
交叉引用（链接图标）打开右侧交叉引用检查面板
导出将当前文档导出为 Word 文件（.docx）并下载

格式面板

点击顶部"格式"按钮，右侧打开格式面板。格式面板的作用是：告诉系统这篇文档应该符合什么排版标准。只有设置好规范并保存，之后的质量检查才会对照格式要求发现不符合的地方。

格式面板分两个子标签：自定义（手动填写）和 AI 智能（AI 自动读取）。

自定义标签 — 手动填写排版要求

预设模板

最上方有模板下拉菜单，可以直接加载内置模板，选中后下方所有字段自动填入，无需手动输入：

模板名称适用场景
海大毕业论文广东海洋大学本科毕业论文格式要求
尖草坪区公文太原市尖草坪区政府公文格式要求
通用实验报告通用实验报告格式

如果保存过自己的模板，会出现在"我的模板"分组中，同样可以直接加载。

页面设置

字段说明
纸张A4 / A3 / B5 / Letter，大多数文档选 A4
上 / 下页面上下边距，单位厘米（cm）
左 / 右页面左右边距，单位厘米（cm）

标题样式

可以分别为一级到四级标题设置：

字段说明
字体下拉黑体 / 宋体 / 楷体 / 仿宋 / 微软雅黑 / Arial / Times New Roman
字号下拉从初号到八号，选中文字号后自动换算为磅值
编号格式标题前的编号样式，如"1"、"1.1"、"1.1.1"，留空表示不要编号

正文样式

字段说明
西文字体文中英文、数字的字体，常用 Times New Roman 或 Arial
中文字体文中中文文字的字体，常用宋体
字号正文文字的大小
首行缩进每段第一行缩进几个字符，通常填 2（符合中文排版习惯）
行距选择模式（倍数 / 固定值）后填入数值；常见如"倍数 1.5"表示 1.5 倍行距
对齐两端对齐 / 左对齐 / 居中 / 右对齐，正文通常选两端对齐

保存为检查规范（绿色按钮）

填好所有需要检查的字段后，点击这个绿色按钮保存。保存成功后会显示"✓ 已保存规范，QA 检查时自动校验"。之后运行质量检查时，系统会自动对照这套规范检查文档格式。

没有填的字段系统会跳过，不会产生误报。只需要填你确实想检查的项目。

+ 另存为模板（小按钮）

将当前填好的格式规范保存为个人模板，下次检查同类型文档时可以直接加载，不需要重新填写。

AI 智能标签 — 自动读取排版规则

不想手动填写？让 AI 帮你读出格式要求。

上传参考模板区域

点击"点击上传参考模板"区域，上传一份已包含格式说明的文档（支持 .docx 和 .pdf），AI 会自动提取其中的排版规范。

快速选择标签

如果没有参考文档，可以点击下方的快速标签（学术论文 / 政府公文 / 实验报告 / 海大论文），自动填入通用描述，再点击"AI 解析排版规则"按钮。

描述输入框

也可以直接用中文描述格式要求，例如：

A4 纸，上下边距 2.5cm，左右 3cm。一级标题黑体三号居中，正文宋体小四首行缩进 2 字符，行距 1.5 倍。

AI 解析排版规则（橙色按钮）消耗 AI 额度

点击后 AI 开始分析，完成后解析结果显示在下方。结果可以手动编辑修改。

保存为检查规范（绿色按钮，解析完成后出现）

确认解析结果正确后点击保存，之后的质量检查会自动使用这套规范。

质量检查面板

点击顶部"检查"按钮（感叹号图标），右侧打开质量检查面板。

运行检查

面板底部有两个检查按钮：

规则检查 免费
本地引擎即时完成，不消耗 AI 额度。检查内容：

• 错别字 — 常见错别字 + 的/地/得
• 数据一致性 — 数字、日期、单位前后是否矛盾
• 标点规范 — 空格、重复标点、括号配对
• 格式合规 — 字体、字号、行距是否符合已保存的规范
🤖 AI 深度 消耗额度
在规则检查基础上，由 AI 进一步分析：

• 论证逻辑 — 因果关系是否成立
• 结论有效性 — 结论是否有充分依据
• 语法优化 — 表述是否清晰流畅
• 需要语义理解的高阶问题

查看和处理结果

检查统计

面板顶部显示四个数字：

数字标签含义
共计发现的全部问题数量
错误需要修改的明确错误（红色）
警告可能有问题，需人工判断（黄色）
提示建议性改进，不强制修改（蓝色）

筛选器

面板提供两个下拉筛选菜单，可以只看某类问题：

  • 按严重程度：全部 / 仅错误 / 仅警告 / 仅提示
  • 按类别：全部 / 错别字 / 数据不一致 / 逻辑问题 / 格式问题

问题列表

每条问题显示描述和原文位置。每条问题右侧有三个操作按钮：

按钮作用
定位中间预览区自动跳转到该问题所在位置，方便对照原文查看
接受接受系统建议的修改，自动替换文中内容
拒绝忽略这条问题，将其从列表中移除

批量操作栏

问题出现后，列表上方会出现三个批量按钮：

按钮作用
✓ 全部接受一键接受所有问题的系统建议修改
✗ 全部拒绝一键忽略全部问题
↩ 撤销采纳撤销刚才"全部接受"的操作，恢复到接受前状态

交叉引用面板

点击顶部"交叉引用"按钮，右侧打开交叉引用面板。

什么是交叉引用？

文档中常见"如图 1-1 所示"、"见表 3"、"参见第二章"这类引用。如果图/表编号修改了但引用文字没改，或者引用了一个不存在的编号，就是引用错误——这类错误在长文档中很难手动检查。

统计数字

标签含义
目标文中找到的图/表/公式/章节总数（被引的对象）
引用文中"如图 X"、"见表 Y"这类引用点的总数
有效引用与目标正确匹配的数量（绿色）
悬空引用了一个找不到对应目标的编号，需要修改（红色）
未引某图/表/章节存在，但全文没有地方引用它（橙色）

三个子标签

标签名内容
引用目标列出扫描到的所有目标（图/表/公式/章节），显示类型、编号、标签、标题
匹配结果列出所有引用点及其与目标的匹配状态（有效/悬空）
可采纳引用系统建议可以添加或修正的引用，可逐条确认处理

运行交叉引用检查（紫色按钮）免费

点击后系统扫描整篇文档，自动填充三个子标签的内容。完成后，重点关注"悬空"（红色数字）的引用——这些是需要修改的错误。

导出 Word

点击顶部"导出"按钮，系统按照格式面板中设定的排版规范重新生成 Word 文件（.docx）并下载到本地。

  • 中文字体（宋体、黑体等）和西文字体（Times New Roman 等）均正确嵌入，在 Word 中打开不会乱码
  • 如果格式面板未设置，导出的文件保留原始文档格式
  • 上传的是 PDF 则导出为 PDF，Word 文件导出为 .docx

常见问题

上传文件后文档显示不出来

请确认文件格式是否在支持列表中（.docx 推荐）。旧版 .doc 文件在某些情况下无法直接打开，建议先在 Word 中另存为 .docx 格式再上传。

格式检查运行了，但没有格式相关的问题

格式检查需要先在格式面板填好排版要求，并点击绿色"保存为检查规范"按钮，之后运行规则检查才会对照格式校验。如果没有保存，系统只检查错别字和标点。

AI 功能提示"额度不足"

免费账户每日 AI 调用有上限，次日 0 点自动恢复。如果需要更多额度，可升级付费套餐解除限制。

接受了修改后发现改错了，怎么撤销

点击中间编辑工具栏的"↩ 撤销"按钮，或按 Ctrl+Z，可以逐步撤销操作。

同一套格式规范想用于多篇文档

在格式面板填好规范后，点击 "+ 另存为模板"，输入名称保存。下次打开新文档后，从模板下拉菜单选择该模板加载，再运行检查即可。

检查结果中有些问题感觉是误报

直接点击该问题右侧的"拒绝"按钮，将其从列表移除，系统不会再提示该条。规则检查对专业术语或特殊表达可能偶有误报，逐条处理即可。
